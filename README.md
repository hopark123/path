## 🚘 설치방법 & 실행방법
```c
git clone https://github.com/hopark123/path.git 
cd pathedu && npm run start
```

## ❗ 실행 에러 해결방법
```c
rm -rf package-lock.json yarn.lock
cd pathedu && rm -rf package-lock.json yarn.lock && npm install
npm run start
```
